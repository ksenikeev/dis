Блок-чейн информации о стратегиях и результатах недельных изменений стоимости корзины валют

В соответствии с заданием вам необходимо выбрать 4 валюты для формирования корзины недельных запасов.
Реестр является носителем информации о выбранных валютах и расчитанной матрицы игры.

Система строится на полном и безоговорочном доверии владельцу реестра (:\)

Добавление информации в реестр

Информация добавляется путем отправки GET запроса на ресурс http://188.93.211.195/currency/add?value={"name": …} с параметром value через котороый передаются данные (URLEncoded !!!).
Формат данных:
{"name": "ФИО студента, номер группы", "currency1": "Код валюты 1", "currency2": "Код валюты 2", "currency3": "Код валюты 3", "currency4": "Код валюты 4", "strategy": "Код стратегии", "x11":" … ", … , "x44":" … ", "p11":" … ", … , "p44":" … " }

Возможные значения currency1 (2,3,4): {AUD, CAD, EUR, USD, SGD, JPY, DKK, CHF, GBP, NOK, NZD, HKD, CNY, BYR, HUF}

Возможные значения strategy: {s1,s2,s3,s4}

В качестве разделителя целой и дробной частей в параметрах x_ij, p_ij используется точка

Информацию необходимо добавить не позднее 23:59 29.11.2021 г.

У каждого студента набор валют должен быть уникальным (порядок имеет значение, т.е. {EUR,USD} и {USD,EUR} разные наборы)

С содержимым блок-чейна можно ознакомиться по ссылке

Блок-чейн в формате JSON доступен по ссылке
